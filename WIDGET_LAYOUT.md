# Widget Top ( Hero Design )

# Content 
Image: User data > put in Data Field = latest_Poster

Title: Text > User Data > put in Data Field = latest_Title

Subtitle 1: > Text > User Data > put in Data Field = latest_Subtitle

Subtitle 2: > Text > User Data > put in Data Field = latest_Genres

# Widget Bottom ( Collection Design )

# Content
Item #1 Image: User Data > episode_Thumbnail | Name > Custom String > Last Episode | Description > User Data > episode_Show

Item #2 Image: User Data > movie_Poster | Name > Custom String > Last movie | Description > User Data > movie_Title

Item #3 Image: User Data > latest_Poster | Name > Custom String > Latest Activity | Description > User Data > latest_Title

Item #4 Image: Appilcation Asset > put any image you like for rating (im using ios star png) | Name > Custom String > TMDB Rating | Description > User Data > latest_Rating

# Add Widget Preview ( Hero Design)

# Content
Hero Image: Image > Application Asset > whatever image you like (i used trakt icon png)

# Mini Profile ( Hero Stat Design )

# Content
Stat: Label ON > Text > Custom String > Content: Trakt | Text > Custom String > Content: Stats

Hero Image: Image > whatever image you like (again, i used trakt icon png)

# Activity Accessory ( Stat Design )

# Content
Stat: Text > Custom String > Content: Trakt Status | Icon: Application Asset > whatever image you like (used trakt icon png)
